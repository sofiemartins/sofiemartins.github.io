unsigned long int getMVM_cpu ( )

Number of times the dirac operator was applied to the CPU field data copy of a spinor field. Note, that this is internally implemented to count every application to a half spinor (either odd part or even part) and then divided by two. An application to a spinor defined on the full lattice is internally counted as two operations and then divided by two.

Returns
unsigned long int Number of applications of the Dirac operator to the full lattice
